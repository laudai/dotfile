## Addtion Keybinding
### Terminal
* ctrl+a j	:focusNext Terminal
* ctrl+a k	:focusPrevious Terminal
* ctrl+a h	:focusPreviousPane
* ctrl+a l	:focusNextPane
* ctrl+a ctrl+k	:kill the active Terminal

### Minimap
* shift+alt+m	:toggle the Minimap
